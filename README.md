# CommunicationSurprise_DataCode


This repository contains the data and code necessary to reproduce the results and figures presented in our manuscript on communication surprise data analysis. Below you will find a description of the directory structure and its contents.

## Repository Structure

- `300ppi/` - This directory contains high-resolution versions of all figures and charts as they appear in the manuscript.
- `Comparative Evaluation of Model Performances/` - This directory includes scripts and data for comparing different computational models as described in our study.
- `Model-based PDR Analysis/` - Scripts and data for the analysis of pupil dilation response using the computational models can be found here.
- `Model-based EEG Analysis/` - Contains the code and data for the EEG analysis conducted with the model.
- `Participants-generated and Model-simulated .../` - This directory contains both the raw data generated by participants and the simulated data from our model.
- `manuscript figures/` - All figures used in the manuscript are stored here, ready for publication.
- `RData/` - Stores R data files which hold datasets used for analysis in the `.RData` format.
- `Rhistory/` - R history files capturing the sequence of commands used during the analysis.
- `.gitattributes/` - Defines attributes per path in the repository and helps in handling the line endings across different operating systems.
- `README.md` - The file you are currently reading.
- `manuscript_figures.ai` - Adobe Illustrator files for the figures, providing the source vector graphics.

## Usage

To replicate the analysis and figures from our manuscript:

1. Clone this repository to your local machine using `git clone`.
2. Navigate to the specific directory for each analysis or figure you wish to reproduce.
3. Follow the instructions contained within each directory's specific README file (if applicable) for detailed steps on how to run the code.

## Dependencies

The analysis scripts are predominantly written in MATLAB (88.3%) and R (11.7%). Ensure you have the latest versions of these software installed on your machine to run the scripts.
